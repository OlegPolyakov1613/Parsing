# Парсинг сайта торгов

## Описание проекта

Данный проект представляет собой парсер для сайта торгов [torgi.org](https://torgi.org/). Программа извлекает информацию о лотах (название, цена, ссылка) с HTML-страницы, сортирует их по убыванию цены и позволяет фильтровать результаты по заданному диапазону цен.

## Откуда берутся данные

Данные извлекаются со страницы открытых торгов всех типов:
```
https://torgi.org/index.php?class=Auction&action=List&mod=Open&AuctionType=All
```

Программа:
1. Скачивает HTML-страницу с сайта
2. Парсит таблицу с лотами используя BeautifulSoup
3. Извлекает название, цену и ссылку для каждого лота
4. Сортирует лоты по убыванию цены
5. Позволяет фильтровать лоты по указанному диапазону цен

```bash
git clone https://github.com/OlegPolyakov1613/Parsing.git
cd Parsing
```

### Шаг 2: Создание виртуального окружения
```bash
python -m venv venv
```

### Шаг 3: Активация виртуального окружения

